Validation Tools

This page provides documentation for the validation tools included with the UAE Real-Estate AI Platform. These tools help ensure that your implementation is correct and follows best practices.

Introduction to Validation Tools

The UAE Real-Estate AI Platform includes several validation tools to help you verify that your implementation is correct. These tools check various aspects of your implementation, including:

  • Flow JSON structure and syntax
  • MCP server functionality
  • Database schema and connectivity
  • API integration
  • Overall system health

Using these tools regularly during development helps catch issues early and ensures that your implementation follows best practices.

Flow Validation Tool

The Flow Validation Tool checks the structure and syntax of your AI Flow JSON files to ensure they are valid and can be executed by the Windsurf Editor.

Usage

To validate a flow JSON file, run the following command: 

node scripts/validate-flows.js path/to/your/flow.json

To validate all flows in a directory: 

node scripts/validate-flows.js path/to/your/flows/directory
