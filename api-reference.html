API Reference

This document provides a comprehensive reference for the UAE Real-Estate AI Platform APIs, including both REST and GraphQL specifications.

REST API Specification

The UAE Real-Estate AI Platform provides a comprehensive REST API for integrating with the platform's features. The API follows RESTful principles and uses JSON for request and response payloads.

Base URL

https://api.realestate-ai.ae/v1

Authentication

All API requests must be authenticated using OAuth 2.0 with JWT tokens. Include the token in the Authorization header: 

Authorization: Bearer {your_jwt_token}

Property Endpoints

List Properties

GET /properties

Retrieves a list of properties with optional filtering.
